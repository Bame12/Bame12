## 🔭 Currently Building

### 🏗️ SIAI — Site Intelligence AI
**AI-powered construction site monitoring system.** My B.Tech capstone.

A hub-and-spoke computer vision pipeline that watches construction sites in real time, detecting PPE compliance, safety violations, and site activity — then syncing everything into Odoo ERP for project management.

```
Architecture:  Raspberry Pi 5 (spoke) → FastAPI Hub → Odoo ERP
Model:         YOLOv8s (custom-trained, deployed on edge hardware)
Transport:     Hub-and-spoke, REST callbacks to Odoo milestones
Integration:   Odoo 17 JSON-RPC · project.task · project.milestone
Live demo:     "House 842 Mahalapye" construction project
```

### 🛰️ FMD Predictor — Foot-and-Mouth Disease Outbreak ML
**ML-based predictive modeling for FMD outbreaks in Botswana** using BOTSAT-1 satellite data. My FMD thesis, supervised by Dr. Otlhapile Dinakenyane.

```
Data source:  BOTSAT-1 satellite imagery · epidemiological records
Architecture: 3-tier — Python/FastAPI · PostgreSQL/PostGIS · Ensemble ML
Models:       Random Forest · XGBoost · Neural Networks
Spatial:      PostGIS for geo-referenced outbreak risk zones
Outcome:      Predictive risk maps for livestock movement corridors
```
